AI Question Generator - Prompt Guide
Universal Prompt Template
Use this prompt with any AI (ChatGPT, Claude, Gemini, etc.) to generate quiz questions:

PROMPT:
Generate [NUMBER] multiple choice questions about [TOPIC] with [DIFFICULTY] difficulty level.

Requirements:
- Each question must have exactly 4 answer options
- Only one option should be correct
- Include a brief explanation for each answer
- Make questions engaging and educational
- Avoid ambiguous or trick questions

Format your response as a JSON array with this exact structure:

[
  {
    "text": "The question text goes here?",
    "options": ["Option A", "Option B", "Option C", "Option D"],
    "correct_answer": "Option B",
    "explanation": "Brief explanation of why this is correct.",
    "category": "[TOPIC]",
    "type": "text_mcq"
  }
]

Return ONLY the JSON array, no additional text or markdown code blocks.

Validation Checklist
Before using generated questions, verify:

✅ Each question has exactly 4 options
✅ Correct answer exactly matches one option (case-sensitive)
✅ Explanations are clear and educational
✅ No duplicate questions
✅ Options are roughly similar length
✅ No obvious pattern in correct answers (all B's, etc.)
✅ Questions are factually accurate
✅ Language is appropriate for target audience


Troubleshooting
If AI returns markdown code blocks:
Add to prompt: "Do NOT wrap response in json or  markers"
If options aren't consistent:
Add to prompt: "All 4 options must be similar in length and detail level"
If correct_answer doesn't match options:
Add to prompt: "The correct_answer value must EXACTLY match one of the strings in the options array, including capitalization and punctuation"
If getting explanations that are too long/short:
Add to prompt: "Each explanation should be 1-2 sentences (20-40 words)"

Quick Reference
Minimum Viable Prompt:
Generate [N] MCQ questions about [TOPIC] with [difficulty] level.
Format: JSON array with fields: text, options (array of 4), correct_answer, explanation, category, type.
Return ONLY JSON.
Recommended Full Prompt:
Generate [N] multiple choice questions about [TOPIC] with [DIFFICULTY] difficulty.

Each question needs:
- text: Clear question
- options: Array of exactly 4 strings
- correct_answer: Must exactly match one option
- explanation: 1-2 sentence explanation
- category: "[TOPIC]"
- type: "text_mcq"

Return ONLY a JSON array, no markdown or extra text.
